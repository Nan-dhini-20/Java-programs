# Java Programs 🚀

This repository contains a collection of core Java programs written for learning, practice, and interview preparation.

The programs focus on building strong fundamentals in Java programming.

---

## 📌 Contents

The repository includes Java programs related to:

- Basic Java syntax  
- Data types  
- Conditional statements  
- Loops  
- Number-based programs  
- Pattern programs
- Ecxeption handling
- Logical problem solving  
- Beginner-level algorithms  

Each program is written in a simple and understandable way, suitable for beginners.

## ▶ How to Run the Programs

### 1. Clone the repository
```bash
git clone https://github.com/Nan-dhini-20/Java-programs.git
```

### 2. Navigate to the project folder
```bash
cd Java_Programs
```

### 3. Compile the Java file
```bash
javac FileName.java
```

Example:
```bash
javac ATM.java
```

### 4. Run the program
```bash
java FileName
```

Example:
```bash
java ATM
```
